# Test Scope

## Objective

Validate the main user-facing workflows of the SauceDemo e-commerce application through manual testing and Cypress end-to-end automation.

## In Scope

- Valid and invalid login scenarios
- Product inventory display
- Product sorting
- Adding and removing products
- Shopping cart content
- Checkout information validation
- Order overview
- Successful checkout completion
- User logout

## Initial Test Environment

- **Application:** SauceDemo
- **Environment:** Public demo environment
- **Primary browser:** Google Chrome
- **Automation tool:** Cypress
- **Test language:** JavaScript

## Out of Scope

- Backend and API testing
- Database validation
- Real payment processing
- Performance and load testing
- Security testing
- Accessibility testing
- Mobile application testing
- Full cross-browser testing

## Limitations

- SauceDemo is a public demonstration application.
- Requirements are inferred from observable behavior.
- Test data and application availability are controlled by a third party.
- The initial automation scope targets Google Chrome only.
